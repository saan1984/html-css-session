Flex direction and wrap

  • Flexbox direction can be arranged as row, column, row-reverse, column-reverse
  • we can wrap the items to next line using flex-wrap attribute.

Example1: Column wise

John
Max
Smith

Example2: Row wise

John
Max
Smith

Example3: Row in reverse

John
Max
Smith

Example4: Column in reverse

John
Max
Smith

Example5: wrap

John
Max
Smith
  • example1, example2, example3, example4 is about flex direction
  • example5: Flex items will all try to fit onto one line. You can change that and allow the items to wrap as needed with this property.
