Basics of CSS (Cascading Style Sheets)

  • Here we'll cover the styles - Display, Border, Width, Padding, Margin, Float (left, right, clear)
  • Inline Styles - The CSS styles that are applied to one element using the style attribute.
  • Class Styles - The CSS styles that are applied to one or more element using the class attribute.
