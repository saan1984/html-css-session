Variable & Data Types

  • Javascript variable can hold many data types like number, string, array, object etc.
  • var is used to create a variable.
  • JavaScript has dynamic types. The same variable can be used as different type.

Example 1

  • When adding a number and a string, JavaScript will treat the number as a string.
  • JavaScript evaluates expressions from left to right.
  • If the first operand is a string, all operands are treated as strings.
