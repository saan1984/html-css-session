Basic HTML Tags

  • We'll cover Basic HTML Tags here like b, i, sub, sup, div, span, p etc.
  • p is block level element and should be used for creating blocks of paragraph text.
  • div is block level element and should be used for wrapping sections of a document.
  • span is inline element and should be used for wrapping small portion of text.
